Get Start

1. clone the project
2. run npm install
3. setup your postgres DB and seup your .env file with following variables
   LOCALHOST_DB_CONNECTION=postgres
   LOCALHOST_HOSTNAME=
   LOCALHOST_DB_NAME=
   LOCALHOST_USERNAME=
   LOCALHOST_PASSWORD=
   LOCALHOST_PORT=5432
   DIALECT=postgres

4. npx sequelize-cli db:migrate

5. follow the route set and test each of the endpoint
